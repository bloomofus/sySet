# 材质、纹理与贴图

好的，这是一个非常基础且核心的3D图形学概念。我用一个简单易懂的比喻和详细的解释来帮你彻底分清它们。

想象一下你要装饰一个**白模蛋糕（3D模型）**：

- **纹理 (Texture)**： 就像是**原材料**。它是一张最基础的、**二维的图片文件**。比如，一张木头的照片、一张砖墙的照片、一张带有图案的布料的PNG或JPG文件。它本身没有任何特性，只是一张图。

  - **文件格式**： `.jpg`, `.png`, `.tga` 等图片格式。
  - **核心**： **就是一张图**。

- **贴图 (Map)**： 指的是**使用纹理的方式和目的**。你把这张“木头纹理”图片，**映射 (Map)** 到蛋糕表面上，这个过程就叫贴图。但更重要的是，在3D软件/引擎中，“贴图”特指这张纹理图片**具体用来表现模型的哪种属性**。

  - 所以我们不常说“一张纹理”，而是说：
    - **漫反射贴图 (Diffuse Map / Albedo Map)**： 使用一张纹理来定义模型的基本颜色和图案。
    - **法线贴图 (Normal Map)**： 使用一张特殊的蓝紫色纹理来模拟表面的凹凸细节，欺骗光线，让平坦的面看起来有起伏，但模型实际还是平的（节省性能）。
    - **高光贴图 (Specular Map)**： 使用一张纹理来定义物体表面不同区域的反光强度。
    - **粗糙度贴图 (Roughness Map)**： 定义表面是光滑如镜（黑色）还是粗糙无光（白色）。
    - **金属度贴图 (Metallic Map)**： 定义哪些部分是金属（白色），哪些部分是非金属（黑色）。
  - **核心**： **纹理的应用**。指定了“用哪张纹理”来充当“哪种属性”。

- **材质 (Material)**： 就像是**蛋糕的完整食谱**。它是一个**数据资产**，包含了所有“贴图”和“参数”的集合，并定义了这些元素如何共同作用，最终决定模型看起来是什么样子。

  - 一个材质球里会包含（或引用）：

    - 漫反射贴图
    - 法线贴图
    - 粗糙度贴图
    - 金属度贴图
    - ...等等

  - 此外，材质还包含

    即使没有贴图也能工作的参数

    ，比如：

    - 一个统一的基础色（不用贴图，直接给个红色）。
    - 整体的透明度、自发光强度等。

  - 最重要的是，材质定义了**着色器 (Shader)**。着色器是一段程序代码，告诉电脑的显卡如何计算光线、颜色、阴影等，最终将所有的贴图和参数渲染成你屏幕上看到的效果。

  - **核心**： **最终的视觉效果定义**。是所有贴图和参数的集合，并由着色器程序驱动。

# 用一个表格总结区别

| 特性         | 纹理 (Texture)          | 贴图 (Map)                                 | 材质 (Material)                                              |
| :----------- | :---------------------- | :----------------------------------------- | :----------------------------------------------------------- |
| **本质**     | **数据** - 一张图片文件 | **功能** - 纹理的应用方式                  | **资产** - 所有属性的集合和渲染规则                          |
| **角色**     | 原材料（面粉、糖）      | 原材料的用途（糖用来甜味，面粉用来主体）   | 完整的食谱                                                   |
| **内容**     | 像素颜色信息            | 指向一张纹理，并指定其用途（如：用于法线） | 包含多个贴图输入和各种参数（颜色、数值等）                   |
| **依赖关系** | 独立存在                | 依赖于纹理                                 | 依赖于纹理和贴图（或直接使用参数）                           |
| **举例**     | `Wood.jpg`              | **法线贴图**槽里使用了 `Wood_Normal.jpg`   | **木材材质**，其中包含了：漫反射贴图=`Wood.jpg`，法线贴图=`Wood_Normal.jpg`，粗糙度=0.7 |

------

# 工作流程举例（在UE5中）：

1. **准备纹理**：你在Substance Painter或Photoshop里制作并导出了三张图片：
   - `T_Brick_BaseColor.png` (漫反射)
   - `T_Brick_Normal.png` (法线)
   - `T_Brick_Roughness.png` (粗糙度)
2. **导入UE5**：将这些`.png`文件导入到UE5的内容浏览器。此时，它们被称为**纹理**。
3. **创建材质**：右键单击 -> 材质 -> 命名为 `M_Brick` 并打开。
4. **连接贴图**：在材质编辑器中：
   - 拖入 `T_Brick_BaseColor` 纹理，连接到 **基础颜色 (Base Color)** 输入节点。这时，这张纹理就被用作**漫反射贴图**。
   - 拖入 `T_Brick_Normal` 纹理，连接到 **法线 (Normal)** 输入节点。这时，这张纹理就被用作**法线贴图**。
   - 拖入 `T_Brick_Roughness` 纹理，连接到 **粗糙度 (Roughness)** 输入节点。这时，这张纹理就被用作**粗糙度贴图**。
5. **应用材质**：将保存好的 `M_Brick` **材质**资产拖放到你的3D墙体模型上。模型就呈现出了砖墙的视觉效果。

**总结一下**：
 ​**纹理**是图片，​**贴图**是纹理的用法，​**材质**是包含所有贴图和参数并最终决定物体外观的“总配方”。
