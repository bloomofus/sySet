## 1）镜像（Image）相关命令

| 功能         | 命令                                | 示例                                       |
| ------------ | ----------------------------------- | ------------------------------------------ |
| 查看镜像列表 | `docker images` / `docker image ls` | `docker images`                            |
| 拉取镜像     | `docker pull`                       | `docker pull nginx:latest`                 |
| 构建镜像     | `docker build`                      | `docker build -t myapp:1.0 .`              |
| 删除镜像     | `docker rmi` / `docker image rm`    | `docker rmi nginx`                         |
| 镜像打标签   | `docker tag`                        | `docker tag myapp:1.0 myrepo/myapp:latest` |
| 查看镜像详情 | `docker inspect`                    | `docker inspect nginx`                     |
| 清理无用镜像 | `docker image prune`                | `docker image prune -a`                    |

------

## 2）容器（Container）相关命令

| 功能             | 命令                            | 示例                  |
| ---------------- | ------------------------------- | --------------------- |
| 运行容器         | `docker run`                    | `docker run -d nginx` |
| 查看运行容器     | `docker ps`                     | `docker ps`           |
| 查看所有容器     | `docker ps -a`                  | `docker ps -a`        |
| 启动容器         | `docker start`                  | `docker start web`    |
| 停止容器         | `docker stop`                   | `docker stop web`     |
| 重启容器         | `docker restart`                | `docker restart web`  |
| 删除容器         | `docker rm`                     | `docker rm web`       |
| 强制删除容器     | `docker rm -f`                  | `docker rm -f web`    |
| 查看容器详情     | `docker inspect`                | `docker inspect web`  |
| 查看容器资源占用 | `docker stats`                  | `docker stats web`    |
| 容器重命名       | docker rename 旧容器名 新容器名 |                       |

## 3）进入容器 / 调试排查

| 功能             | 命令          | 示例                       |
| ---------------- | ------------- | -------------------------- |
| 进入容器执行命令 | `docker exec` | `docker exec -it web bash` |
| 查看容器日志     | `docker logs` | `docker logs -f web`       |
| 查看容器进程     | `docker top`  | `docker top web`           |
| 查看容器端口映射 | `docker port` | `docker port web`          |

## 4）网络（Network）相关命令

| 功能         | 命令                        | 示例                                  |
| ------------ | --------------------------- | ------------------------------------- |
| 查看网络列表 | `docker network ls`         | `docker network ls`                   |
| 创建网络     | `docker network create`     | `docker network create mynet`         |
| 删除网络     | `docker network rm`         | `docker network rm mynet`             |
| 查看网络详情 | `docker network inspect`    | `docker network inspect mynet`        |
| 容器加入网络 | `docker network connect`    | `docker network connect mynet web`    |
| 容器退出网络 | `docker network disconnect` | `docker network disconnect mynet web` |

## 6）文件拷贝（宿主机 ↔ 容器）

| 功能          | 命令        | 示例                               |
| ------------- | ----------- | ---------------------------------- |
| 宿主机 → 容器 | `docker cp` | `docker cp ./a.txt web:/tmp/a.txt` |
| 容器 → 宿主机 | `docker cp` | `docker cp web:/tmp/a.txt ./`      |

## 9）Docker Compose 常用命令

| 功能           | 命令                        | 示例                           |
| -------------- | --------------------------- | ------------------------------ |
| 启动服务       | `docker compose up`         | `docker compose up -d`         |
| 停止并删除容器 | `docker compose down`       | `docker compose down`          |
| 查看服务状态   | `docker compose ps`         | `docker compose ps`            |
| 查看日志       | `docker compose logs`       | `docker compose logs -f`       |
| 重建并启动     | `docker compose up --build` | `docker compose up -d --build` |
